# IO Growth Website - GitHub Pages Version

This is the static website version of the IO Growth business consulting site, optimized for hosting on GitHub Pages.

## Structure

- `index.html` - Main HTML file
- `css/styles.css` - All styling for the website
- `js/main.js` - JavaScript functionality
- `assets/` - Contains all images and other static assets

## Deployment Instructions

1. Create a new GitHub repository
2. Upload all files maintaining the folder structure
3. Go to repository Settings > Pages
4. Select the main branch as the source
5. The site will be published at `https://[username].github.io/[repository-name]/`

## Custom Domain Setup (Optional)

If you want to use a custom domain:

1. In repository Settings > Pages, enter your custom domain
2. Create/update DNS records with your domain provider:
   - Add an A record pointing to GitHub Pages IP addresses, or
   - Add a CNAME record pointing to `[username].github.io`
3. Add a CNAME file to your repository with your domain name

## Local Development

To preview the site locally, simply open the `index.html` file in your browser.
